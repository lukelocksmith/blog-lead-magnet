Flexible CTA system for blog posts with analytics and floating bar.

== Description ==

Blog Lead Magnet lets you create unlimited CTAs and inject them into blog posts
based on configurable display conditions. Track performance with built-in analytics.

Features:

* Flexible CTA builder — add as many CTAs as you need
* Display conditions: after each H2 section, at 30%/50%/70% of article, at the end
* Exclusion rule: one CTA per position, highest priority wins
* Full customization: heading, body, image, shortcode, button, colors, text size
* Analytics: impressions, clicks, CTR per CTA
* Floating bar with delay, dismiss, position settings
* Auto-updates from GitHub releases

== Installation ==

1. Upload the `blogcta` folder to `/wp-content/plugins/`
2. Activate the plugin through the Plugins menu
3. Go to Blog Lead Magnet in the admin sidebar
4. Add your first CTA
